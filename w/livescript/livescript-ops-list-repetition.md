List repetition is a type of list [operator](livescript-ops) in Livescript. It uses the same <code>*</code> operator as number multiplication.

<div class="codeblock">
<pre class="leftcol">
\# LiveScript
[\ha] * 3 #=> ['ha','ha','ha']
</pre>
<pre class="rightcol">
// Generated Javascript
['ha', 'ha', 'ha'];
</pre>
